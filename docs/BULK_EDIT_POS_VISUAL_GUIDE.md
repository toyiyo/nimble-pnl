## Interaction States

### State 1: Default (Read Mode)
```
Button: [Select]
Checkboxes: Hidden
Action Bar: Hidden
Panel: Closed
```

### State 2: Selection Mode (No Items Selected)
```
Button: [Done]
Checkboxes: Visible (all unchecked)
Action Bar: Hidden
Panel: Closed
```

### State 3: Selection Mode (Items Selected)
```
Button: [Done]
Checkboxes: Visible (some checked)
Action Bar: Visible with count
Panel: Closed
Selected Cards: Blue ring + background
```

### State 4: Categorizing
```
Button: [Done]
Checkboxes: Visible (some checked)
Action Bar: Visible
Panel: Open (slides from right)
Background: Dimmed overlay
```

### State 5: Applied
```
Button: [Select] (auto-exits selection mode)
Checkboxes: Hidden
Action Bar: Hidden
Panel: Closed
Toast: Visible with undo (10s)
Cards: Show new category
```
